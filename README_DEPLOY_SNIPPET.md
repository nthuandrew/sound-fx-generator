## Deployment Instructions

1. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```

2. Set environment variables (e.g., Google API Key in `.env` if needed)

3. Start the Flask server
   ```bash
   python web_app.py
   ```

4. Open your browser and go to [http://localhost:5000](http://localhost:5000) to use the web interface

> To change the port, modify the `port` parameter in `web_app.py`.
